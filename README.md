# Kerex Nigeria — Website

All code for the Kerex corporate site lives in this folder.

## Folder layout

```
kerex-website/
  index.html          ← page shell
  package.json        ← dependencies
  vite.config.js      ← dev server / build
  public/             ← static assets
  src/
    main.jsx          ← entry point
    App.jsx           ← page layout
    styles.css        ← all CSS
    scroll.js         ← scroll animations
    components/       ← React sections
```

## Run

```bash
cd C:\Users\HP\Documents\kerex-website
npm install
npm run dev
```

Open http://localhost:5173
